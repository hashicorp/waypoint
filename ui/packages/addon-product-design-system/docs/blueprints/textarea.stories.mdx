# Textarea

## Example

```html
<div>
  <label for="control">
    Field Name
  </label>

  <p id="help">
    Help text goes here.
  </p>

  <textarea
    id="control"
    name="..."
    aria-describedby="help errors"
  ></textarea>

  <ul id="errors">
    <li>Error message 1</li>
    ...
    <li>Error message N</li>
  </ul>
</div>
```

## Prerequisites

Please review the following information:

* ["Forms: Form Fields"](?path=/docs/guides-forms-overview--page#form-fields)

## Elements

Field elements should appear in the following order:

1. Field Name
2. Help Text
3. Form Control
4. Field Errors

For the majority of use cases, this can be achieved by defining
markup in explicit DOM order.
