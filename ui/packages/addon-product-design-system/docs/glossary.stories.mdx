# Glossary

## Assistive Technology (AT)
Technologies that aid UI interactivity for disabled users (e.g., screen readers, switch devices, etc.)

## ARIA
["Accessible Rich Internet Applications"](https://www.w3.org/TR/wai-aria/)

## Form Control
Native form control widgets (e.g., `<input>`, `<select>`, and `<textarea>`)
