# A11y Mixins


## visuallyHidden

- remove border, outline, and shadows
- reduce dimensions to absolute minimum without removing element from accessibility tree
- prevent interruption of normal document flow
- render foreground and background content invisible

```sass
@use "pds/utils/a11y";

span {
  @include a11y.visuallyHidden;
}
```
