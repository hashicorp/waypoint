# Visually Hidden

The `.pds--visuallyHidden` class will render an element "invisible" to sighted
users, while keeping its content in the accessibility tree for use by
assistive technologies like screen readers.


In your stylesheet:
```sass
@use "pds/helpers/visuallyHidden";
```

In your markup:
```html
<span class="pds--visuallyHidden">...</span>
```
