# Help Text

A visual style for blocks of text that provide additional context to the user.

- [Markup](#markup)
- [Styling](#styling)
- [Accessibility](#accessibility)
- [See Also](#see-also)


## Markup

Apply the `.pds-helpText` class to a `<p>` element.

```html
<p class="pds-helpText">...</p>
```


## Styling
Styles are scoped to the `.pds-helpText` selector.

```scss
@use "pds/components/helpText";
```
