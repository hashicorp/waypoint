# Field Name

A visual style to communicate the name of a field or fieldset.

- [Markup](#markup)
- [Styling](#styling)
- [Accessibility](#accessibility)
- [See Also](#see-also)


## Markup

1. Apply the `.pds-fieldName` class to a semantic `<label>` or `<legend>`.

```html
<div class="pds-formField">
  <label class="pds-fieldName">...</label>
  ...
</div>

<fieldset class="pds-formField">
  <label class="pds-fieldName">...</label>
  ...
</fieldset>
```


## Styling
Styles are scoped to the `.pds-fieldName` selector.

```scss
@use "pds/components/fieldName";
```
