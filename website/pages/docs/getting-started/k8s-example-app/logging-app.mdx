# The `waypoint logs` Command

From the same application directory, you can execute the `waypoint logs` command to inspect the running logs for your deployment.

```bash
waypoint logs
```

You should see an output similar to the below. You'll note that we are seeing continuous probe checks against the root location. This is
from our `probe_path` value in the original `waypoint.hcl` file!
