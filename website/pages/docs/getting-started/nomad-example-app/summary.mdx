# Walkthrough Summary

You have now successfuly deployed the example NodeJS application on Kubernetes with Waypoint!

In this guide, you accomplished:

- Initializing Waypoint for an example application.
- Building the application using a Cloud Native Buildpack with the the `pack` plugin
- Updating the application to use the `docker` plugin, as well as specific configurations for NodeJS a port.
- Updating the application and then redeploying it.
- Executing commands with `waypoint exec`.
- Viewing the application's Waypoint log.
- Accessing the web UI.
