# Prerequisites

## Examples Repository

This walkthrough uses an example Ruby application to show you how to build, deploy and release using Waypoint.

This example application is part of the [Waypoint Examples applications](https://github.com/hashicorp/waypoint-examples). You will need a local clone of this repository on your computer to complete this walkthrough.

Clone the Waypoint Examples repository and navigate to the Ruby example directory.

```shell
$ git clone https://github.com/hashicorp/waypoint-examples.git

$ cd waypoint-examples/ruby
```

## Docker for Desktop

Install the most recent stable version of [Docker for Desktop](https://www.docker.com/products/docker-desktop).
