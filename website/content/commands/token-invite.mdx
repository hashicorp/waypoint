# Waypoint Token invite

Command: `waypoint token invite`

Request a new invite token.

@include "commands/token-invite_desc.mdx"

## Usage

Usage: `waypoint token invite [options]`

#### Global Options

- `-workspace=<string>` - Workspace to operate in.

#### Command Options

- `-lifetime=<duration>` - How long the invite token will valid for, starting now.
