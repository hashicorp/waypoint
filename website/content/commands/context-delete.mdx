# Waypoint Context delete

Command: `waypoint context delete`

Delete a context.

@include "commands/context-delete_desc.mdx"

## Usage

Usage: `waypoint context delete [options]`

#### Global Options

- `-plain` - Plain output: no colors, no animation.
- `-app=<string>` - App to target. Certain commands require a single app target for Waypoint configurations with multiple apps. If you have a single app, then this can be ignored.
- `-workspace=<string>` - Workspace to operate in.

#### Command Options

- `-all` - Delete all contexts. If this is specified, NAME should not be specified in the command arguments.
