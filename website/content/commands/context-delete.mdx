# Waypoint Context delete

Command: `waypoint context delete`

Delete a context.

@include "commands/context-delete_desc.mdx"

## Usage

Usage: `waypoint context delete [options]`

#### Global Options

- `-workspace=<string>` - Workspace to operate in. Defaults to 'default'.

#### Command Options

- `-all` - Delete all contexts. If this is specified, NAME should not be specified in the command arguments.
