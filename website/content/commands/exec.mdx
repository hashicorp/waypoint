# Waypoint Exec

Command: `waypoint exec`

Execute a command in the context of a running application instance.

@include "commands/exec_desc.mdx"

## Usage

Usage: `waypoint exec [options]`

#### Global Options

- `-workspace=<string>` - Workspace to operate in. Defaults to 'default'.
