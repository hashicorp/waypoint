# Waypoint Config get

Command: `waypoint config get`

Get config variables.

@include "commands/config-get_desc.mdx"

## Usage

Usage: `waypoint config get [options]`

#### Global Options

- `-workspace=<string>` - Workspace to operate in. Defaults to 'default'.

#### Command Options

- `-json` - Output in JSON
- `-raw` - Output the value for the named variable only (disables prefix matching)
- `-app=<string>` - Scope the variables to a specific app.
