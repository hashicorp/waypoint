# Waypoint Ui

Command: `waypoint ui`

Request a new invite token.

@include "commands/ui_desc.mdx"

## Usage

Usage: `waypoint ui [options]`

#### Global Options

- `-workspace=<string>` - Workspace to operate in. Defaults to 'default'.

#### Command Options

- `-authenticate` - Creates a new invite token and passes it to the UI for authorization
