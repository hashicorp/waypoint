# Deploy an Example Application with Docker

This walkthrough runs on your computer with Docker for Desktop and will give you a very quick overview of how Waypoint works.

- **Time required:** About 10 minutes.
- **Prerequisites:** [Docker for Desktop](https://www.docker.com/products/docker-desktop) and a copy of the [Waypoint Examples applications](https://github.com/hashicorp/waypoint-examples).
