# Deploy an Example Application with Docker

This walkthrough runs on your computer with Docker for Desktop and will give you a very quick overview of how Waypoint works. If you would like to see how Waypoint deploys apps to Kubernetes, try the [Kubernetes walkthrough](/docs/getting-started/k8s-example-app).

- Time required: About 10 minutes
- Prerequisites: Docker for Desktop and the Waypoint Examples applications
