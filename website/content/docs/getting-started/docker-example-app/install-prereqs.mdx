# Prerequisites

<!-- Make this a partial -->

## Examples Repository

You will need a local clone of the source code repository for the [Waypoint Examples applications](https://github.com/hashicorp/waypoint-examples) on your computer. You will use one of these apps to learn how to use Waypoint to deploy an app, update it, and then redeploy.

Clone the repository.

```shell
$ git clone https://github.com/hashicorp/waypoint-examples.git
```

## Docker for Desktop

Install the most recent stable version of [Docker for Desktop](https://www.docker.com/products/docker-desktop).
