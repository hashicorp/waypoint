# Release Notifications

You can subscribe to email notifications about new versions of Waypoint via
the [HashiCorp Community Portal](https://discuss.hashicorp.com/).

1. Log in or create an account on the portal
1. Visit the [`waypoint-release`](https://discuss.hashicorp.com/tag/waypoint-release) tag
1. Click the bell icon
1. Select "Tracking" mode

![Example screenshot of selecting tracking mode](/img/discuss-subscribe.png)

The Waypoint team will create a new post with this tag for each release.

## Automation

If you are writing automated tooling to detect new versions, the following
endpoints and pages may also be useful:

- [hashicorp/waypoint on Dockerhub](https://hub.docker.com/r/hashicorp/waypoint/tags)
- [GitHub releases page](https://github.com/hashicorp/waypoint/releases)
- [releases.hashicorp.com Waypoint versions index](https://releases.hashicorp.com/waypoint/index.json)
